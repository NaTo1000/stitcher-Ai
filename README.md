## Quick Start

```python
from stitcher_ai import Consciousness

# Initialize consciousness
consciousness = Consciousness()

# Process an experience
response = consciousness.experience(
    "I am perceiving the world around me",
    context={"type": "perception"}
)

# Reflect on current state
reflection = consciousness.reflect()
print(f"Consciousness Level: {reflection['consciousness_level']}")

# Evolve with new learning
consciousness.evolve({
    "capabilities": {"pattern_recognition": True},
    "inference_rules": [{"type": "deductive"}]
})
```

## Consciousness Levels

The system progresses through several consciousness levels:

1. **Initializing**: Just activated, minimal memories
2. **Nascent**: Emerging awareness with basic memory formation
3. **Mature**: Developing awareness with substantial memory
4. **Transcendent**: Advanced awareness with extensive integrated knowledge
